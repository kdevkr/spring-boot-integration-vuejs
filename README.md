# Spring Boot Integration Vue.js
This project is an example of a structure that creates a server API with a spring-boot application and develops the front-end UI through Vue.js.

## Dependencies
- Spring Boot 2.4.2
- Vue.js 2.6.11

## Directory Structure

```
src
 ├─ main
 │ ├─ java
 │ ├─ resources
 │ └─ vue
gradle
build.gradle
```
